# Android-IM
## Android 即时通信app
### 简介
项目利用[环信](http://www.easemob.com/)sdk实现了即时通信的单聊、群聊等。
### 功能
1. 环信sdk的集成及欢迎页面
2. 登录页面
3. 主页面
4. 设置页面
5. 联系人列表页面
6. 添加好友
7. 邀请页面
8. 会话页面
9. 主页面中的会话页面
10. 群列表页面
11. 新建群页面
12. 群联系人选择页面
13. 群详情
